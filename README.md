# 造成リスク概算ツール GitHub Pages公開用

このフォルダは、造成リスク概算ツール Ver.1.0 試作4相当をGitHub Pagesで公開するためのフォルダです。

不動産査定・買取打診前の初期検討で、造成費の概算レンジと見落としやすい造成リスクを確認するためのローカル保存型Webアプリです。正式な工事見積ではありません。実際の売買判断、買取価格判断、事業収支判断では、施工業者・造成業者・設計者等による見積確認が必要です。

## 公開用フォルダの内容

- `index.html`
- `style.css`
- `app.js`
- `data.js`
- `README.md`

## キャッシュ対策

GitHub Pages公開時のキャッシュ対策として、`index.html` で以下の読み込みクエリを付与しています。

```html
<link rel="stylesheet" href="style.css?v=10p4">
<script src="data.js?v=10p4" defer></script>
<script src="app.js?v=10p4" defer></script>
```

## 保存キーとデータ保存

localStorage保存キーは変更していません。

```text
zouseiCostLocal.data.v1
```

GitHub Pages公開後も、案件データと単価設定は端末・ブラウザごとのlocalStorageに保存されます。クラウド保存ではありません。

- PCとスマホで自動同期はしません。
- 別ブラウザ間でも自動同期はしません。
- ブラウザデータ削除により保存データが消える可能性があります。
- GitHub PagesのURL変更や公開パス変更により、以前のlocalStorageデータが見えなくなる可能性があります。
